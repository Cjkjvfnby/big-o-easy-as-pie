Code samples

Simple operations

a = 1  # O(1)

a += 1  # O(1)

b = [1, 2, 3]  # O(n)

b[0]  # O(1)


for x in b:  # O(n)
    print(x, end=" ")  # O(1)

min(b)  # O(n)

d = [x + 1 for x in b]  # O(n)

Sequential operations is added

def foo(a: list):
    for x in a:  # O(n)
        ...

    for y in a:  # O(n)
        ...

This code complexity is O(n) + O(n) -> O(n)

Nested is multiplied

def foo(a: list):
    for x in a:  # O(n)
        ...
        for y in a:  # O(n)
            ...

This code complexity is O(n) * O(n) -> O(n2)
